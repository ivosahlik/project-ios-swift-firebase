# CocoaPods

## Add Firebase SDK

### Create a Podfile if you don't have one:
pod init

### Open your Podfile and add:
pod 'Firebase/Core'

https://firebase.google.com/docs/auth/ios/custom-auth
pod 'Firebase/Auth'

### Save the file and run:
pod install
